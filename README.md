# rs.swb.admin

> 个人后台管理系统

## 已开发页面

- 登录页

## 已有功能

- 路由跳转
- Bs 全局对象
  - config: 项目基础配置。
  - Authorization: 用户授权信息，登录后由后端返回。
  - sessionStorage: 设置、获取 sessionStorage。
  - clearSessionStorage: 用于删除、清空 sessionStorage 的值。
  - messageBox: 消息确认框。
  - htpp: 发送 post 请求。

## 已有组件

- input（输入框）
- button（按钮）
- collapse-transition（展开收起）
